# 📊 Customer Segmentation Using RFM Analysis | Power BI Dashboard

> An interactive Business Intelligence dashboard built in **Power BI** to analyze customer behavior using the **RFM (Recency, Frequency, Monetary) model**. The dashboard helps businesses identify valuable customer segments, monitor revenue performance, and support data-driven marketing strategies.

---

## 📌 Project Overview

Customer segmentation is a key business strategy that enables organizations to understand customer purchasing behavior and deliver personalized marketing campaigns.

This project applies the **RFM (Recency, Frequency, Monetary)** framework to classify customers into meaningful segments such as **Champions**, **Loyal Customers**, **Potential Loyalists**, **At Risk**, and **Lost Customers**.

The dashboard provides an interactive experience through dynamic slicers, KPIs, and visualizations, allowing users to analyze customer performance across multiple business dimensions.

---

## 🎯 Business Objectives

- Analyze customer purchasing behavior using the RFM model.
- Identify high-value and at-risk customers.
- Monitor monthly revenue trends.
- Compare revenue across countries, territories, occupations, and genders.
- Enable interactive filtering for business decision-making.

# 🧠 RFM Segmentation

| Metric | Description |
|---------|-------------|
| **Recency** | Number of days since the customer's last purchase |
| **Frequency** | Number of purchases made by the customer |
| **Monetary** | Total amount spent by the customer |

Customers are segmented into:

- Champions
- Loyal Customers
- Potential Loyalists
- At Risk
- Lost Customers
